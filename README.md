# Gen Z Gender Political Polarization

GitHub Pages source for an English-language public companion to Simone Ghezzi Colombo's master's thesis:

*Masculinities, Feminist Mobilization and the New Cultural Divide: A Quantitative Analysis of Gendered Political Attitudes Among Young Europeans*.

## Structure

- `index.html`: main page
- `styles.css`: layout and visual styling
- `app.js`: regime selector and CSV-driven detail cards
- `assets/figures/`: original SVG thesis figures used on the site
- `assets/maps/`: regime map
- `data/`: exported CSVs used by the interactive cards
- `thesis.pdf`: full dissertation

## Pages

This repository is intended to be published with GitHub Pages from the root of the `main` branch.
